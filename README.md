**Geotrek mobile**, mobile app of *Geotrek Rando* (http://geotrek.fr).

DEVELOPMENT
===========

This project is generated using Yeoman (http://yeoman.io/)

### Requirements
- npm (version > 1.2.10)
- grunt (`npm install -g grunt-cli`)

    - `$ npm install`
    - `$ bower install`
    - `$ grunt serve`
    - look your browser
